Cybersecurity Awareness Chatbot
Description of project
This a C# console based chatbot designed to help and educate users about basic cybersecurity practices and how to stay safe online. The chatbot interacts with users and provides them with information on topivs such as passwords, phishing and safe browsing
Features
Voice greeting using a WAV audio file
Ascii logo display from an image
user input with validation
chatbot interaction using loop based responses
coloured sonsole interface for better user experirence
Example topics you can ask:
Password safety
Phishing
safe browsing
or general and basic cybersecurity questions
